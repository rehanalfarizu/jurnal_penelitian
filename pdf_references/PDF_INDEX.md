## Cara Mengutip di Paper

Untuk paper, gunakan format Vancouver/IEEE dengan nomor dari `references.md`:

```
[9]   Author, "Title", Journal/Conf., Year.
```

Atau jika mau mengutip PDF tambahan (bagian B), gunakan prefix `[A]`:
```
[A4] Author, "Efficient GPT-4V...", Nature Communications, 2025.
```
